# My to-do app deployed in AWS 

This simple app is to practice deployment using S3 host and CloudFront CDN of AWS and GitHub Actions to auto deploy.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
